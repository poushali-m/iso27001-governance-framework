# ISO 27001:2022 Information Security Policy & Governance Framework

An information security policy and governance framework developed for Bluefrontier, aligned with ISO/IEC 27001:2022 Clauses 5.2 and 5.3.

## Academic Context

Developed as part of the Enterprise Security Management module at the University of West London. Demonstrates practical application of ISO 27001:2022 governance requirements in a real-world organisational scenario.

## Overview

This framework provides Bluefrontier with a structured approach to managing information security, covering policy objectives, governance roles, escalation procedures, KPIs, and legal compliance mapping.

## What This Framework Covers

- Information Security Policy aligned to Clause 5.2
- Governance structure and organisational roles (Clause 5.3)
- Three-level escalation process for security incidents
- Phased implementation plan across 6 phases
- Key Performance Indicators (KPIs) with measurable targets
- Legal and regulatory mapping — ISO 27001, UK GDPR, Data Protection Act 2018
- Real-world phishing incident case study with lessons learned
- Security awareness and training programme
- Risk management communication plan

## Key Roles Defined

- Chief Information Security Officer (CISO)
- Security Governance Committee
- ISMS Manager
- Information Security Officer (ISO)

## Standards and Regulations Referenced

- ISO/IEC 27001:2022
- UK General Data Protection Regulation (UK GDPR)
- UK Data Protection Act 2018
- NIS Regulations 2018
